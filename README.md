# ServerWatch Agent

Backend service for ServerWatch.

ServerWatch Agent collects operational information from a home server and exposes it through a simple REST API for companion clients such as the Wear OS application.

## Features

- Storage monitoring
- Service health monitoring
- qBittorrent integration
- REST API
- systemd deployment
- Lightweight ASP.NET Core Minimal API

## Current Integrations

- qBittorrent
- Jellyfin
- Sonarr
- Radarr
- Prowlarr
- Immich

## Companion Projects

- ServerWatch Wear OS
